# D-Kafka

## Description
D-Kafka is a super-easy-to-start dockerized Kafka cluster with Zookeeper for development environment

## Usage

1. Start the container:
```shell
docker-compose up
```
2. Navigate to [localhost:9092](http://localhost:9092)


## Commands

- start the container:
```shell
docker-compose up
```
- stop the container:
```shell
docker-compose down
```
